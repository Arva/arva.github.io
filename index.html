Responsive Animation Platform for building mobile, desktop, smart TV's and web applications.

Animations

Granular animations have never been easier to code. Using the Layout and Flow APIs, any animation is possible.

Cross platform

Whether you want to deploy your idea on mobile, desktops, smart TVs, websites or all at once, ARVA has you covered.

Responsive by nature

With scalability in mind, you can build UIs that effortlessly adapt to all possible screen sizes.

Native feel

Delight your customers with a great native-like experience, complete with 60fps granular animations.

Enterprise-grade quality code

We know how managing sizeable projects can turn into a nightmare, so we built ARVA to fit your most ambitious goals.

Component reusability

You never enjoyed reinventing the wheel. Neither did we. And with ARVA, you don't have to, either.


ARVA Connects

It has never been as important to build your app with realtime in mind. ARVA Connects with Firebase, SharePoint and REST APIs.


Building blocks

ARVA is built on top of awesome technologies. It uses jspm.io for packet management, famo.us for crazy 60fps animations, famous-flex to make famo.us usable for grownups.


Supported by

ARVA is documented with ESDoc, GitHub Wiki and tested using Browserstack. We embrace their services and support to the open-source community.
